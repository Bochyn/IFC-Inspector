# IFC Inspector Documentation

Welcome to the IFC Inspector documentation.

## Overview

IFC Inspector is a terminal-based tool for exploring IFC (Industry Foundation Classes) files. It provides fast, lightweight inspection of BIM models without requiring heavy CAD software.
